运行方式

查看一下版本号是否正确

  flutter --version
  
运行以下命令查看是否需要安装其它依赖项来完成安装

  flutter doctor
  
运行启动您的应用

  flutter packages get 
  
  flutter run
  
如果有其他问题,请参考

https://flutterchina.club/setup-macos/

https://flutter.dev/docs/get-started/install/macos
